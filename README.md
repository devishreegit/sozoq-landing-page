# sozoq-landing-page
Single page landing website for Sozoq internship task built using WordPress and Elementor.

## Project Overview

This project is a responsive landing page designed to showcase a company's services and portfolio.
It was built using **WordPress** with the **Elementor page builder**.

## Features

* Hero section with title, tagline, and call-to-action button
* About section describing the company
* Services section highlighting core offerings
* Projects / Work showcase with sample projects
* Functional contact form
* Footer section

## Contact Form

The contact form collects the following information:

* Name
* Email
* Phone Number
* Message

Form submissions are handled using the **SureForms plugin**, and entries are stored in the WordPress dashboard.

## Technologies Used

* WordPress
* Elementor
* SureForms
* HTML / CSS (via Elementor styling)

## Project Structure

```
wp-content/
 ├── themes
 ├── plugins
 └── uploads
```

## How to Run the Project

1. Install **XAMPP** or any local server.
2. Place the project inside the `htdocs` directory.
3. Import the WordPress database if required.
4. Start Apache and MySQL.
5. Open the project in a browser.
