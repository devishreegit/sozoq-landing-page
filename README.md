# Sozoq Landing Page

Single page landing website for Sozoq internship task built using WordPress and Elementor.

---

## Project Overview

This project is a responsive landing page designed to showcase a company's services and portfolio. It was built using WordPress with the Elementor page builder.

---

## Features

- Hero section with title, tagline, and call-to-action button  
- About section describing the company  
- Services section highlighting core offerings  
- Projects / Work showcase with sample projects  
- Functional contact form  
- Footer section  

---

## Contact Form

The contact form collects the following information:

- Name  
- Email  
- Phone Number  
- Message  

Form submissions are handled using the SureForms plugin, and entries are stored in the WordPress dashboard.

---

## Technologies Used

- WordPress  
- Elementor  
- SureForms  
- HTML / CSS (via Elementor styling)  

---

This project is developed using WordPress. The repository includes documentation and screenshots to demonstrate the implementation.
